## 🚀 Quick Command Reference

### Backend
```bash
# Setup (first time)
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run
python main.py
# Server on http://localhost:8000

# Test
curl http://localhost:8000/api/health
```

### Extension
```bash
# Setup (first time)
cd extension
npm install
npm run compile

# Watch mode (during development)
npm run watch

# Run (press F5 in VS Code)
# Opens Extension Development Host

# Test extension commands
Cmd+Shift+P → "Ciper"
```

### Full Stack
```bash
# Terminal 1: Ollama
ollama serve

# Terminal 2: Backend
cd backend
source venv/bin/activate
python main.py

# Terminal 3: Extension (in VS Code)
cd extension
npm run watch
# Press F5 to debug
```
