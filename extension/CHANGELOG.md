# Changelog

All notable changes to Ciper Agent will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [0.2.0] - 2026-04-09

### Added
- **Persistent chat history** — conversations are now saved to SQLite and survive restarts
- **Export chat** — export any session as a Markdown file (`Cmd+Shift+E`)
- **Search history** — search within chat session via QuickPick
- **Code syntax highlighting** — AI responses render fenced code blocks with a Copy button
- **Markdown rendering** — bold, italic, headings, lists rendered in the chat panel
- New backend endpoints: `/api/sessions`, `/api/chat/{id}/history`, `/api/chat/{id}/export`, `/api/chat/{id}/search`
- Session ID is now tied to workspace folder name (isolated per project)

### Changed
- Backend version bumped to `0.2.0`
- Improved webview UI: auto-resize textarea, streaming indicator border, tighter layout
- `chat_engine.py` now stores model name alongside each message

### Fixed
- Redundant `activationEvents` removed from `package.json` (auto-generated by VS Code)

---

## [0.1.0] - 2026-04-09

### Added
- Initial release
- Chat panel (WebView) with streaming responses
- Ollama model switcher via QuickPick + status bar
- Commands: Open Chat, Analyze Code, Generate Plan, Switch Model, Clear History
- Right-click context menu: Analyze Code, Generate Plan
- Keyboard shortcuts: `Cmd+Shift+C/A/L/M`
- FastAPI backend with endpoints: health, models, chat (streaming), plan, analyze-code
- Ollama client: generate, chat (multi-turn), pull model
- In-memory context manager (upgraded to SQLite in v0.2.0)
- File context injection (language, filename, selected text)
- VS Code theme-aware colors throughout
